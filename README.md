# daily-scheduler
Schedule your today to make your tomorrow better

## how to compile
- Download the files (or clone it with `git`)
- Install `cargo` and run:
```
cargo build --release
```
And then check `target/release/daily-scheduler`.

## colors available
- BLACK
- RED
- GREEN
- YELLOW
- BLUE
- MAGENTA
- CYAN
- WHITE

## default config file
```
[colors]  
description = BLUE
hours = GREEN  

[database]  
path = ./data.db  
```
